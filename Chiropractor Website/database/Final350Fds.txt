Assumptions:
There will be only one doctor.
There will only be one office manager.
PNo will never get deleted from the patient's table.
You can get any city and state with a zip code and address.
You can get any group number and expiration date with an InsName and MemberID
You will never be able to be in a room without an appointment.
Two people will never have the same email and password.
Two people will never have the same first name and date of birth.
PNo is unique
DNo is unique
OmNo is unique
A SOTAP, does not need a respective attribute on creation
Every patient created will have default OTAP values.

DNo -> DName, DDob, DAge, DGender, DAddrs, DZip, DPhone, DMail, DPassword, DCity, DState
